📖 About
This project was developed to understand advanced encryption techniques by building a secure encryption tool. The goal was to implement real-world data protection mechanisms using cryptographic methods.

🚀 What This Project Does
This tool allows users to encrypt and decrypt files securely using AES-based encryption. It supports multiple keys, password protection, and hidden storage of encryption keys to enhance security.

🛠️ Tools & Technologies
Python
Cryptography Library (Fernet – AES based)
Hashing (SHA-256)

📂 Project Files
advanced_encryption_tool.py → Main encryption and decryption tool
keys/ (hidden folder) → Stores encryption keys securely
README.md → Documentation

▶️ How to Run
Install required library:
pip install cryptography

Run the program:
python advanced_encryption_tool.py

Select options from menu and perform encryption or decryption

📊 Output
The tool successfully encrypts and decrypts files while maintaining data security. Files become unreadable after encryption and are restored correctly after decryption using the correct key and password.
